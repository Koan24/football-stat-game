# Football Stat Game

A full-stack football statistics game in which players select footballers
and seasons that satisfy specific requirements while trying to achieve
the highest possible score.

## Project status

The project is currently in the initial development stage.

## Main concept

Each game contains five answer slots. Every slot has a set of football-related
requirements and one scoring category.

The user must:

1. select a footballer,
2. select one of their seasons,
3. satisfy all slot requirements,
4. maximize the value of the scoring statistic.

## Planned features

- footballer search,
- season selection,
- dynamic qualifiers,
- player-season validation,
- score calculation,
- percentile ranking,
- daily challenges,
- responsive user interface,
- light and dark mode,
- game result sharing.
